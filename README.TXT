для того чтобы развернуть мое прилоджение сначала нужно:
1) установить Ansible последней версии
2) в файле hosts.yml указать свои данные: адрес целевой машины, имени пользователя для подключения по SSH,
пароль для подключения
3) склонировать репозиторий на машину где установлен Ansible
4) перейти в папку репозитория
5) запустить плейбук с помощью команды "ansible-playbook main.yml" если пароли нету зашифроманных данных с помощью
ansible-vault или если есть шифрованные данные то используем команду, "ansible-playbook main.yml --ask-vault-pass"
после чего нужно будет ввести пароль для расшифровки

Дествия после того как плейбук отработает:
ваш сайт будет доступен по протоколу HTTP, http://<адрес сервера на который було все установлено>
